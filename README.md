# VK Compute

A lightweight Vulkan-based computing engine for matrix operations.

## Requirements

- **GCC** (or compatible C compiler)
- **Make**
- **Vulkan SDK** (installed and configured on your system)

## Quick Start

```bash
# Clone the repository
git clone https://github.com/hazeldiv/vk-compute

cd vk-compute

# Clean previous builds
make clean

# Build and run
make run
```
